Data Structure Visualizer

The app lets you visualize the data flow of the selected data structure.

The JavaScript code is provided by you; the visualization is accessible via a simple API.

UI

The code editor is a bastardized version of ACE.

The controls below the editor, from left to right: toggle help window; toggle console; select data structure; change theme; run code.

The editor, the console and the visualizer area are all draggable, and the editor window is resizable.

API

It's best to think of the API as a glorified version of console.log().

Generally, you need to define the data structure, instantiate it, and call the draw(instanceVariable) method in order to see the default visual output.

Call the paint() method in your algorithm to trace its execution.
